Service First

Service First is a fork of the Active Agenda project. It consists of the basic technology platform and the PHP application generator (spec2app) used to convert XML module specs (like e.g. Buildings) to a browser application atop a MySQL database. The GUI is inspired by the idea of displaying structured content frequently encountered in organizations as a blog. Each detailed view can be annotated by notes, links, tags and reminders. The application can be used with similar look and feel on desktops and tablets. The technology platform installation is prepared to be run and administered on top of cloud platforms like Microsoft Azure. Modules can be installed on the run. The scripts handle all the tasks on the application server and database level preventing conflicts from data structure and data content mismatch. The small technology footprint allows economical sizing of virtual machines.

A domain application can incorporate several hundreds of modules. Different parties can have access to the same module with different access rights and use the module for sharing information between organizations.
